# Let's Build: GPT Mini Studio Workflow

This document explains the step-by-step process of how this project was built, along with code snippets and explanations. This guide is designed to help you explain the architecture to others!

## 1. Environment Setup
First, we created an isolated Python virtual environment to manage dependencies without affecting the global system, and installed PyTorch, FastAPI, and other libraries.
Snippet (`run.sh` / terminal):
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt fastapi uvicorn pydantic
```
